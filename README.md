# Bindi Maps Assignment

Description : 


BindiMaps Apps is on tracking of a path that someone has travelled and will also show the venues inside of the buildings. 

**The first screen** will give you the list of users with travelled paths. 

**On the second screen,** it will show you the travelled path of the particular user and also will show you the venues of the building. 

<br>

Basic Details : 

- Kotlin
- MVC structure
- Canvas
- Draw Line for user sessions and circles for Venues
